# MyCommonUi

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 7.3.0.

## First Start

```
npm i && npm run build && npm i && npm run start
```

Navigate to `http://localhost:8000/`

```
npm run build && npm i && npm run start
```
